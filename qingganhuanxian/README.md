- 防晒服

- 昼夜温差大，带件秋装外套

2、洗漱品：

- 防晒霜

- 其他

2、药品类:

- 鼻炎药：鼻炎康片？

> 西北天气干燥，可能会对鼻炎不太友好，最好带着鼻炎药

- 营养物质：维生素、葡萄糖

- 感冒药：布洛芬

- 肠胃药：蒙脱石散

- 外用药品：敷贴、创可贴、纱布、碘伏

- 其他：便携氧气瓶

- 墨镜、防沙防水鞋套、帽子、口罩、**棒球棍**

- 大保温壶

- 垃圾袋

4、食品：超时购买

- 红牛、矿泉水

- 水果

- 零食
