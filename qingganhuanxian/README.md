### DAY 2 (06-18)

西宁 — 29KM(2270) —> 塔尔寺(2670) - 136KM -> 青海湖(3200) — 145KM(3700) —> 茶卡盐湖(3100) ——> 茶卡镇(3100)

*注：青海湖到茶卡会穿越一段盘山路，可能会有高反*

青海湖攻略：

- 定位: 最美油菜花海

- 定位：二郎剑（进入景区，只能做船到二郎剑半岛，此处可以近距离接触水边）

- 定位：二郎剑天鹅湖

- 在开往茶卡盐湖的路上寻找花田

茶卡盐湖攻略：

- 需要做小火车来回进入核心景区，走的话，路有点长

- 火车来回需要单独购票

途经加油站：

早餐：看宾馆

午餐：二郎剑景区附近

晚餐：茶卡镇

入住酒店(06-18 -> 06-19)：

- [茶卡盐湖青盐2号宾馆]:https://hotels.ctrip.com/hotels/detail/?hotelId=66784866: 景区内，可以看日出、日落和星星

美食推荐：

[茶卡镇美食](https://www.weelv.com/qa/10460/119666.html)

可以摘星星的地方：

- 茶卡盐湖

 > 攻略：建议备好干粮，下午5点之前买票进入，游客开始往外面走，这时可以先观看茶卡的晚霞。然后沿着小火车的铁轨一直往里面走，越到里面约干净拍出的效果也更好。或者也可以选择在入口处的盐雕处拍摄，盐雕和星空相接的景色。

B计划：
    
    如果在青海湖游玩的时间太晚，可以 DAY3 再去茶卡盐湖

### DAY 4 (06-20)

```
大柴旦镇(3185) —12KM—> 大柴旦翡翠湖(3150) —115KM—> 南八仙魔鬼城(2761) —394KM/(3650)—> 敦煌市(1140)    
                  |
                  | (大环线，这个有点累)
                  V
                 U 型公路 —95KM—> 东台吉乃尔湖(观景台) —23KM—> 乌苏特水上雅丹 —42KM—> 西台吉乃尔湖 —159KM—> 南八仙魔鬼城 —107KM—> 大柴旦镇
```

*注：*

- 魔鬼城到敦煌会穿越祁连山余脉，海拔最高3650m，可能会有高反

- 从翡翠湖到魔鬼城经过G215国道，车少，可以航拍沙漠之路

- 从魔鬼城到敦煌
    
    - 穿越*当金山*，215国道 -- 最美道路之一，也是最危险道路之一
 
    - 走G3011柳格高速，走隧道穿越当金山

途经加油站：

早餐：宾馆

午餐：在翡翠湖景点吃，或者自带，魔鬼城是无人区

晚餐：敦煌夜市

入住酒店(06-20 -> 06-22)：

- [敦煌云天国际酒店(沙洲夜市店)](https://hotels.ctrip.com/hotels/detail?hotelid=5398509&cityid=11)

可以摘星星的地方：

- 大柴旦翡翠湖

> 攻略： 37°星空营地，应该西北最有名的看星空地点了，但除了37°还有一个野奢帐篷营地

B计划：
    
    若 DAY3 有时间可以先去翡翠湖，DAY4 到敦煌有时间的话，可以先去月牙泉，这样话，DAY5 就只玩莫高窟，算是休整一下。

## 携带物品

1、衣服：

- 防晒服

- 昼夜温差大，带件秋装外套，或者轻薄羽绒服

- 多带双鞋

2、洗漱品：

- 防晒霜

- 其他

2、药品类:

自带的：

    - 鼻炎药：鼻炎康片？

        > 西北天气干燥，可能会对鼻炎不太友好，最好带着鼻炎药

    - 营养物质：维生素

    - 感冒药：布洛芬

    - 肠胃药：蒙脱石散

    - 外用药品：敷贴、创可贴、纱布、胶布

当地购买：

- 便携氧气瓶、葡萄糖、碘伏棉签

3、装备类：

- 墨镜、防沙防水鞋套、帽子、口罩、毛巾、导汗条

- 大保温壶

- 一次性雨衣

4、物资：超市购买

- 红牛、矿泉水、牛奶

- 纸巾、湿巾

- 水果

- 零食(不要买膨胀包装的食品，可能会“嘣~”)

- 红糖

- 预备自热食品，两餐的量即可
